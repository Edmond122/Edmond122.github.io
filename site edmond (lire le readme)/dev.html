Bienvenue sur l'onglet "développement" de notre jeu

Objectifs et Outils

Pour réaliser ce jeu nous avons eu besoin du logiciel CodeBlocks pour le programme, et du logiciel Gimp pour l'utilisation des images. Notre progamme est codé en C. Tout d'abord, voici les bibliothèques qui ont été nécessaires à la réalisation du jeu.

Mon travail

Pour réaliser notre jeu, nous nous sommes répartis le travail afin d'être plus efficaces

Mes principales tâches ont été:

- Trouver et appliquer les images

- Les monter afin de faire le fond du jeu

- Créer l'interface du jeu

- Faire des fonctions pour aller chercher, puis afficher les images

- Créer les 2 fonctions de démarrage du jeu

- Créer la fonction pour afficher les boutons, sur lesquels le joueur devra appuyer dans le bon timing pour marquer des points
